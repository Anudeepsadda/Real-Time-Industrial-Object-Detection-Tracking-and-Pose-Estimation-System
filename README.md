# Real-Time Robotic Perception System for Industrial Assembly

An industry-oriented Computer Vision and Robotics project developed for real-time industrial object perception and robotic assembly applications. The system integrates YOLOv8-based object detection, ByteTrack multi-object tracking, Kalman-filter-based motion estimation, and 6D pose estimation using OpenCV ArUco markers and solvePnP.

The project was trained on an industrial object detection dataset and optimized using ONNX for deployment-ready inference. A Streamlit-based web application with ngrok deployment was developed to enable real-time browser-based object detection and visualization.

---

# 🚀 Key Features
- Real-time industrial object detection using YOLOv8
- Multi-object tracking with ByteTrack and Kalman Filters
- 6D pose estimation using OpenCV solvePnP
- ONNX optimization for deployment-ready inference
- Streamlit web application for live testing
- GPU-accelerated inference in Google Colab
